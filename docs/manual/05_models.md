## 모델 관리

현재 프로젝트에 **저장된 모델(★ 저장 아티팩트)** 의 목록·상세·삭제를
관리하고, 예측 페이지로 빠르게 이동합니다.

### 👤 사용자 관점

**할 수 있는 일**

- 저장된 모델 목록 조회 (`FR-075`)
- 모델 상세 — 알고리즘, 학습 데이터셋, 저장 시각, 평가 지표 요약
- "예측으로 이동" 버튼으로 해당 모델 즉시 예측 페이지로 이동
- 불필요한 모델 삭제 (아티팩트·DB 행·감사 로그 포함)

**기본 사용 순서**

1. 카드에서 모델 이름을 클릭해 상세 패널을 펼칩니다.
2. **예측으로 이동** 또는 **삭제** 중 필요한 동작을 선택합니다.

**자주 쓰는 버튼/옵션**

| 버튼 | 설명 |
| --- | --- |
| 예측으로 이동 | `current_model_id` 세션에 저장 + 예측 페이지로 `switch_page`. |
| 삭제 | `models` 행 + `storage/models/<model_id>/` 전체 제거. 예측 이력은 보존. |

**주의사항 & 팁**

- 모델은 **프로젝트 단위** 로만 보입니다 — 다른 프로젝트의 모델은 프로젝트를 전환한 뒤 조회하세요.
- 삭제는 되돌릴 수 없습니다. 예측 이력이 참조하는 경우 이력의 `model_id` 는 남지만, 다시 예측을 수행할 수는 없습니다.
