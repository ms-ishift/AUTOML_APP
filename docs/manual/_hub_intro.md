이 페이지는 AutoML MVP 의 **전체 매뉴얼** 을 한 화면에서 훑을 수 있는 허브입니다.
상단 검색창에 단어를 입력하면 일치하는 섹션만 남습니다 — 예: `CSV`, `베스트`, `libomp`.

- 👤 **사용자** 섹션은 "버튼/입력을 어떻게 쓰는가" 를 안내합니다.
- 🛠 **개발자** 섹션은 "어느 파일·FR 번호와 연결되는가" 를 안내합니다.

각 페이지 상단의 *❓ 이 페이지 도움말* 박스로도 같은 내용을 바로 볼 수 있습니다.
